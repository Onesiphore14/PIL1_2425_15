Description de la conception

Composant Technologies utilisées Description
Base de données SQLite La base de données gère les utilisateurs, leurs données personnelles, les trajets de covoiturage et la messagerie. Elle contient des tables pour enregistrer les informations personnelles, les offres et demandes de trajets, ainsi que les messages échangés. Les relations entre les tables permettent de faire des correspondances entre conducteurs et passagers.
Backend Python (Flask) Le backend gère l'inscription et la connexion des utilisateurs, la mise à jour des profils, la réinitialisation des mots de passe, et fait le lien entre le frontend et la base de données. Il implémente également la logique de matching et la messagerie instantanée.
Frontend HTML, CSS, JavaScript Le frontend comprend plusieurs pages : ifri.html (page d'accueil), premièrepage.html (page de connexion/inscription), et les interfaces pour la gestion des profils, des trajets et de la messagerie. Leaflet.js est utilisé pour l'affichage des cartes et la géolocalisation.
Messagerie WebSockets, JavaScript La messagerie permet aux utilisateurs d'échanger des messages en temps réel grâce à un système de conversation. Chaque échange est stocké dans une table messages liées àune cpnversation unique. Le backend FLASK gère l'envoi et la reception des conversations

Instructions de déploiement

  1. Cloner le dépôt Git du projet
  2. Installer les dépendances nécessaires (Flask, SQLite, etc.)
  3. Lancer l'exécution de app.py pour démarrer le serveur backend
  4. Ouvrir ifri.html dans un navigateur web pour accéder à l'interface
  5. Créer un compte via le formulaire d'inscription
  6. Se connecter avec les identifiants créés
  7. Accéder à son profil pour compléter les informations
  8. En tant que conducteur, publier des offres de trajet
  9. En tant que passager, publier des demandes de trajet

Manuel d'utilisation

1. Première utilisation

Étape 1 : Accédez à l'application en ouvrant ifri.html dans votre navigateur.

Étape 2 : Si vous n'avez pas de compte, cliquez sur "S'inscrire" et remplissez le formulaire.

Étape 3 : Après inscription, connectez-vous avec votre email et mot de passe.

2. Gestion du profil

Étape 1 : Cliquez sur "Mon profil" dans le menu principal.

Étape 2 : Complétez vos informations personnelles (point de départ, horaires, etc.).

Étape 3 : Si vous êtes conducteur, ajoutez les informations sur votre véhicule.

Étape 4 : Sauvegardez les modifications.

3. Publication d'offres/demandes de trajet

Pour les conducteurs :

- Allez dans "Proposer un trajet"

- Renseignez le point de départ, d'arrivée, l'heure et le nombre de places

- Validez la publication

Pour les passagers :

- Allez dans "Chercher un trajet"

- Indiquez vos besoins (départ, arrivée, horaire)

- Validez la demande

4. Utilisation de la messagerie

Étape 1 : Dans la liste des matches, sélectionnez un utilisateur.

Étape 2 : La messagerie s'ouvre automatiquement.

Étape 3 : Tapez votre message et appuyez sur "Envoyer".

Étape 4 : Vous recevrez une notification pour les nouveaux messages.

5. Consultation des trajets

Étape 1 : Allez dans "Mes trajets" pour voir vos offres/demandes en cours.

Étape 2 : La carte montre les trajets disponibles avec les points de départ/arrivée.

Étape 3 : Cliquez sur un trajet pour voir les détails et les options disponibles.
